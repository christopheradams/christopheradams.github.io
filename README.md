# [christopheradams.io](https://christopheradams.io)

## About

A website of articles and photographs by Christopher Adams. Built with [Jekyll].

## Requirements

```sh
gem install bundler
npm install --global webpack
```

## Instructions

```sh
bundle install
npm install
webpack
bundle exec jekyll build --lsi
```
